# Contivo Website

Готовый стартовый сайт Contivo на Astro + Cloudflare Pages.

Что уже есть:

- Главная страница
- Privacy Policy
- Terms of Use
- Help
- Contact
- News
- Логотип Contivo из `public/logo.png`
- Переключение языка: русский / English / Deutsch
- Структура для будущего развития сайта

## Запуск

```bash
npm install
npm run dev
```

Локально сайт откроется обычно по адресу:

```txt
http://localhost:4321
```

## Где менять тексты и ссылки

Основной файл:

```txt
src/content/site.js
```

## Где менять дизайн

```txt
src/styles/global.css
```

## Где лежит логотип

```txt
public/logo.png
```

Если хочешь заменить логотип, просто положи новый файл с таким же именем.
