Куда вставить файлы

Все файлы из этой папки положить сюда:

contivo_website/public/

Должно получиться:

public/
├── robots.txt
├── sitemap.xml
├── site.webmanifest
├── favicon.png
├── icon.png
└── logo.png

После вставки:
1. GitHub Desktop
2. Summary: Add SEO public files
3. Commit to main
4. Push origin
5. Дождаться зелёной галочки в Actions

Потом проверить:
https://contivoapp.com/robots.txt
https://contivoapp.com/sitemap.xml
https://contivoapp.com/site.webmanifest
